# 🏦 Banking Management System

A professional banking web application built with Python Flask.

## ✨ Features

- 🔐 Secure login/register with bcrypt
- 💰 Deposit & withdraw functionality
- 📊 Transaction history
- 🔗 REST API endpoints
- 🎨 Professional UI design

## 🛠️ Tech Stack

- **Backend:** Python Flask
- **Database:** SQLite
- **Security:** bcrypt
- **Frontend:** HTML5, CSS3
